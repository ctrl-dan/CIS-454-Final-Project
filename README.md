
made using
- React (w/ ContextAPI)
- Socket.io 
- Nodejs


- Node.js ([Download](https://nodejs.org/en/))

### Steps
1) Open your terminal (linux/MacOS) or Git Bash.
2) Run ```npm install``` in order to install all the dependancies.
3) Run ```npm start```.
4) Open your browser and go to ```localhost:3000```.
